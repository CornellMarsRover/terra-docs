Detailed Description

A class for waiting for service calls.

Freeing the service listener with del or using with is technically optional, as the garbage collector will do this if you don't.

Example:

def cb(req, resp):
resp.success = True
resp.message = "Starting"
return resp
print("Waiting for trigger")
with ServiceListener(Trigger, f"/{namespace}/wait", cb) as serv:
serv.wait_for_msg()
print("Trigger received, we can start!")
