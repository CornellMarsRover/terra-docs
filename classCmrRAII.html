Detailed Description

template<typename Func>
class CmrRAII< Func >

A RAII helper class that simply provides a way to call a function when it is destroyed.

Meant to be used with the ALWAYS macro.

Example:

{C++}
ALWAYS(this) { this->m_finished = true; };
ALWAYS
#define ALWAYS(...)
Macro for easily creating a CmrRAII object that calls the code when it is destroyed.
