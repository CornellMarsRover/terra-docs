Detailed Description

This node simply provides a topic and action server for testing purposes.

It echos back every message on the topic /<ns>/<name>/test_in on the topic /<ns>/<name>/test_out.

It echos back every target position goal received on the action server at /<ns>/<name>/test_action and responds on the action server /<ns>/<name>/test_return.
