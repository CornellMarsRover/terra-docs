Detailed Description

SiteAnalyze

The SiteAnalyze class controls the collection and analyze tasks for astrotech. It completes several tasks including analyzing, filling sites, gearshifting, and scooping. The class listens for messages of MotorWriteBatch and uses these messages to change aspects of the different motors being used, such as the motor id, mode, and angles.
