Detailed Description

The FaultHandler class provides a service interface to allow nodes to restart themselves if an error occurs.

The service is created on /<namespace>/recover_fault and takes messages of type cmr_msgs::srv::RecoverFault.

The fault handler will activate nodes through the lifecycle manager.
