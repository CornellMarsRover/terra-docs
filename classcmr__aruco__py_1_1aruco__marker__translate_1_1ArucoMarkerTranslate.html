Detailed Description

This node is responsible for translating markers from the /aruco_markers topic (configurable via the "aruco_marker_topic" parameter) into a MarkerArray that can be displayed in RViz or Foxglove Studio.

The marker size can be configured to match the actual value by setting the "marker_size" parameter; the default is 0.2m.
