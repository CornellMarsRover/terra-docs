Detailed Description

Hardware interface for the drives system of the rover, which consists of a number of wheel motors and their encoders.

This hardware interface supports commanding the velocity of each wheel, and reading the current velocity and position of each wheel reported from the wheel encoders. This hardware interface is mainly used by the diff_drive_controller to provide velocity control of the rover.
