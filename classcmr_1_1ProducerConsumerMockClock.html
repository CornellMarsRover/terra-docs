Detailed Description

template<typename ClockType>
class cmr::ProducerConsumerMockClock< ClockType >

A clock that can be mocked for testing.

This clock is designed to work with 1 producer and 1 or more consumers in different threads. The producer is the thread that is setting the time, and the consumers are the threads reading the time.

Set the time and waits until a consumer registers it.

This function is not safe to be called when using multiple producers. One producer could set the time and wait and another producer could overwrite that time before any consumer sees the first time. This would leave the first producer waiting forever.

This function blocks the calling thread
