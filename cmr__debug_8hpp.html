Macro for declaring a callback function for a FabricNode.

This macro essentially wraps a try-catch around the code you provide to turn any assetion errors into calls to error_transition() to restart the node instead of terminating it

This should be used in all service, topic, etc. callback handlers

Example:

{C++}
m_sub = create_subscription<std_msgs::msg::Bool>(
get_name() + std::string("/kill"), 10,
[this](const std_msgs::msg::Bool::SharedPtr msg) {
CMR_RETRY_ON_ERR(RCLCPP_INFO(get_logger(), "Got kill message: %s",
msg->data ? "true" : "false");
CMR_ASSERT_MSG(false, "Killed by kill message");)
});
