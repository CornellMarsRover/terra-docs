Detailed Description

template<typename ServiceT>
class cmr::fabric::LifecycleClient< ServiceT >

Lifecycle Client is just like a rclcpp::Client but it can be activated and deactivated.

When inactive, the client will not be able to send requests.

Template Parameters
ServiceTthe type of the service
