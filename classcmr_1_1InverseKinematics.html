Detailed Description

InverseKinematics

The inverse kinematics node plans and executes arm poses. A base station node sends pose messages (position and orientation) to the node, which then either plans and executes the pose or returns an error if the pose is unreachable. It subscribes to the "/arm_pose_topic" topic for poses and requires MoveIt 2 to be launched to function.
